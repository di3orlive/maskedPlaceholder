# maskedPlaceholder
Placeholder mask for old browsers. include `ie5+` <br/>
Pure js.


Usage
-----
Just add class `"placeholder-mask"` and data attribute `"data-placeholder"` to input.

Like this:
-----
```html
<input type="text" data-placeholder="1111111111" class="placeholder-mask">
```
